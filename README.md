# OpenMindi Studio

OpenMindi Studio is a modern, high-performance web application built for the African tech and creative sectors. It showcases design, AI capabilities, and comprehensive case studies. 

This project is built using:
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS & OpenMindi Design Tokens
- **Animation**: Framer Motion
- **Language**: TypeScript

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5000](http://localhost:5000) with your browser to see the result (note: we use port `5000` via our npm `dev` script in `package.json`).

## Project Structure

- `app/`: Next.js app router structure (pages, layouts, API routes)
- `components/`: Reusable React components (UI elements, layout components, sections)
- `lib/`: Utility functions and shared library code

## Design System

The application uses the strict OpenMindi design system parameters:
- **Typography:** Playfair Display (Serif) for headings, DM Sans (Sans) for body text, Space Mono (Monospace) for utility labels. 
- **Colors:** Clay (`#C1622F`), Ochre (`#D4903A`), Bark (`#2E1A0E`), Dusk (`#1A0F0A`), Sand (`#F5EDD8`), and Cream (`#FAF6EE`).

## Scripts

- `npm run dev`: Starts the local development server
- `npm run build`: Creates an optimized production build
- `npm run start`: Starts the application in production mode
- `npm run lint`: Runs ESLint
